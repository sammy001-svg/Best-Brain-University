# Best Brain International University (BBIU) Website

A premium, responsive website for **Best Brain International University**, a leading higher education institution located in Paynesville, Liberia.

## Features

- **Modern & Premium Design**: Built with a focus on high-end aesthetics, cinematic imagery, and glassmorphism.
- **Comprehensive Academics**: Includes dedicated pages for Colleges, Programs, and Academic Calendars.
- **Admissions Management**: Structured overview for undergraduate and transfer students.
- **Student Life**: Showcasing campus culture, organizations, and vibrant student activities.
- **Dynamic Blogs**: Integrated news and blog section on the home page.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.

## Technologies Used

- **Core**: HTML5, Vanilla CSS3, JavaScript
- **Icons**: Font Awesome 6
- **Typography**: Google Fonts (Playfair Display, Inter)

## Project Structure

- `index.html`: Main home page
- `about-overview.html`, `history.html`, `leadership.html`: University information
- `academics-overview.html`, `programs.html`: Educational offerings
- `student-life.html`: Campus experience
- `assets/`: High-resolution imagery and university branding
